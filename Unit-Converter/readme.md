# Unit Converter

## About 
This is a basic  unit conversion webapp, built in python using fastapi.
This is my personal implementation of the beginner python series that it is available on [roadmap.sh](https://roadmap.sh/projects/unit-converter).

#### Usage
    $ fastapi dev main.py

#### Supported Quantities and their units
* Length
    * millimeter
    * centimeter
    * meter
    * kilometer
    * inch
    * foot
    * yard
    * mile

* Weight
    * miiligram
    * gram
    * kilogram
    * ounce
    * pound

* Temperature
    * Celsius
    * Fahrenheit
    * Kelvin
